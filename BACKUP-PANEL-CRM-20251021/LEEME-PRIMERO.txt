✅ Zona horaria: Argentina (configurable)
✅ Diseño moderno con tema oscuro
✅ Sistema de búsqueda de eventos
✅ Auto-refresh cada 30 segundos

🚀 INICIO RÁPIDO:

1. Lee el archivo: QUICK-START.md
2. Copia la carpeta "panel" a tu proyecto
3. Ejecuta: npm install
4. Configura el archivo .env con tus credenciales
5. Ejecuta: node server.js
6. Abre: http://localhost:3000
